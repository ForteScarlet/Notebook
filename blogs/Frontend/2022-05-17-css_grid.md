# CSS Grid（网格布局）

> 目前为止最强大的`CSS`布局方案

:::demo

<html>
  <div class="grid">
    <div class="child child-1">1</div>
    <div class="child child-2">2</div>
    <div class="child child-3">3</div>
  </div>
</html>

<style>
.grid {
  display: grid;
  grid-gap: 5px;
  width: 100%;
  height: 200px;
}
.child {
  background: #ccc;
}
.child-1{
  grid-column:1;
  grid-row:1;
}
.child-2{
  grid-column:2;
  grid-row:1;
}
.child-3{
  grid-column:2;
  grid-row:2;
}
</style>

:::
